Github

Github es un portal para alojar el código de las aplicaciones de cualquier desarrollador, y que fue comprada por Microsoft en junio de 2018. La plataforma será creada para los desarooladores suban el código de sus aplicaciones y herramientas, y que com usuario no solo puedas descargar la aplicación, sino también entrar a su perfil para leer sobre ella o colaborar con su desarrollo.
